# Домашнее задание / практическая задача

Создайте React-приложение на TypeScript для демонстрации работы props и state на примере корзины товаров.

Проект должен быть создан без Vite. Для сборки можно использовать esbuild.

## Требования:

### 1. Описать тип товара

Создайте тип `Product`, который содержит:

```typescript
type Product = {
  id: number;
  title: string;
  price: number;
};
```

### 2. Создать массив товаров

Создайте отдельный файл с массивом товаров. Каждый товар должен иметь:

- уникальный `id`;
- название;
- цену.

### 3. Создать функциональную версию компонентов

Реализуйте функциональный компонент `ProductCardFunction`, который:

- принимает товар через `props`;
- принимает количество товара через `props`;
- принимает функции увеличения и уменьшения количества через `props`;
- отображает название товара, цену, количество и итоговую стоимость по этому товару;
- содержит кнопки `+` и `-`.

Реализуйте функциональный компонент `CartSummaryFunction`, который:

- принимает через `props` общее количество товаров;
- принимает через `props` общую сумму;
- отображает информацию о корзине.

### 4. Создать классовую версию компонентов

Реализуйте классовый компонент `ProductCardClass`, который делает то же самое, что и `ProductCardFunction`. Компонент должен:

- наследоваться от `Component`;
- получать типизированные `props`;
- обращаться к данным через `this.props`;
- возвращать JSX из метода `render()`.

Реализуйте классовый компонент `CartSummaryClass`, который делает то же самое, что и `CartSummaryFunction`.

### 5. Реализовать состояние в App

В компоненте `App` создайте два независимых состояния.

- Первое состояние должно использоваться только для функциональных компонентов.
- Второе состояние должно использоваться только для классовых компонентов.

То есть:

- изменение количества товара в функциональной версии не должно менять классовую версию;
- изменение количества товара в классовой версии не должно менять функциональную версию.

### 6. Реализовать логику корзины

Нужно реализовать:

- увеличение количества товара;
- уменьшение количества товара;
- запрет ухода количества ниже 0;
- подсчёт общего количества товаров;
- подсчёт общей суммы;
- отдельную кнопку очистки для функциональной корзины;
- отдельную кнопку очистки для классовой корзины.

### 8. Проверить работу приложения

В результате на странице должны быть две независимые секции:

- Functional components;
- Class components.

В каждой секции должен отображаться список товаров и summary корзины.

Если нажать `+` в функциональной секции, должна измениться только функциональная корзина.
Если нажать `+` в классовой секции, должна измениться только классовая корзина.

## Цель задания

Закрепить:

- передачу данных через `props`;
- передачу функций через `props`;
- работу со `state`;
- типизацию `props` в `TypeScript`;
- разницу между функциональными и классовыми компонентами;
- принцип подъёма состояния в родительский компонент;
- работу с независимыми состояниями внутри одного приложения.
